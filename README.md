# OIBSIP — Oasis InfoByte Summer Internship Program
### Track: 🔐 Cyber Security

This repository contains all task submissions for my **Oasis InfoByte Summer Internship Program (SIP)**  
under the **Cyber Security** track.

> ⚠️ **Ethics Notice:**  
> All scanning, exploitation, firewall, and traffic-capture tasks in this repository were performed
> exclusively on systems I own or local virtual machines I control (e.g., Kali Linux, DVWA).
> No external, third-party, or production systems were targeted.

> 💡 While the Cyber Security track only requires **4 of 10** tasks for completion,
> I completed **all 10** to build a stronger portfolio and improve eligibility
> for a **Letter of Recommendation**.
