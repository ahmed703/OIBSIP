# Oasis Infobyte Internship – Cyber Security Track (OIBSIP)

This repository contains cyber security practical tasks completed as part of the
Oasis Infobyte Internship Program.

## Track
Cyber Security

## Tasks Included
- Task 1: Network Scanning (Beginner)
- Task 2: Password Strength Analysis (Beginner)
- Task 3: Phishing Awareness (Beginner)
- Task 4: Keylogger (Beginner)
- Task 5: Packet Sniffing (Beginner)
- Task 6: Password Cracking (Beginner)
- Task 7: Vulnerability Assessment (Intermediate)
- Task 8: Web Application Security Testing (Intermediate)
- Task 9: Intrusion Detection System (Advanced)
- Task 10: Malware Analysis (Advanced)

Each task follows the official OASIS INFOBYTE task guidelines and includes
documentation and practical evidence.
